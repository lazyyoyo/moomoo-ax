# dogfooding/

moomoo-ax의 product loop를 **moomoo-ax 자기 자신에 돌린 결과**.

이 폴더의 문서들(`seed.md`, `jtbd.md`, `problem-frame.md`, `scope.md`, `prd.md`)은 team-ax의 define 단계가 자동으로 생성한 산출물이다.

## 원칙: 실험 결과물 ≠ 프로젝트 정의

- **`PROJECT_BRIEF.md`** — 오너가 쓰고/승인한 프로젝트 정의. SSOT.
- **`dogfooding/`** — product loop가 자동으로 뽑은 실험 결과. 참고용.

dogfooding 결과가 PROJECT_BRIEF로 승격될 수는 있어도 둘은 같지 않다.

## 메타 지표

dogfooding이 `PROJECT_BRIEF.md`와 얼마나 가까운지는 **meta loop의 자기 점검 지표** 중 하나. 큰 gap이 계속 벌어지면:

- define 단계의 script/rubric 개선 신호
- 또는 오너가 프로젝트 방향을 바꿨다는 신호 (이 경우 PROJECT_BRIEF 갱신)
